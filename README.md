# multiple-comfysd
这是另一个项目“comfysd”的可部署网站版，可以给多个人一起用，做了并行优化。

但是为了给多人用，此项目连接了2台云端实例，本地的话也是两台。因为只有一个显卡一个实例会让两个模型抢显存。
